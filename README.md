# 🍽️ La Casa Restaurant Website

La Casa is a modern restaurant landing page designed to showcase a restaurant’s atmosphere, menu, and dining experience.  
The website focuses on clean design, strong visual presentation, and a user-friendly layout suitable for restaurants and food businesses.

## 🚀 Live Demo
https://v0-la-casa-restaurant-website.vercel.app/

## 📌 Features
- Modern and elegant restaurant landing page
- Hero section with call-to-action
- Menu presentation for dishes and specials
- Reservation section
- Responsive layout for desktop and mobile devices
- Clean and minimal user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Project Purpose
This project was created as part of a front-end development portfolio to demonstrate the ability to design and build modern websites for restaurants and hospitality businesses.
